What do the states mean?

Foghorn can list these states:

  1. Open
    The port is not blocked in any way and accessible for everyone on the public internet. This means connections will always succeed, regardless of who connects.
  2. Consistent
    The publically visible port number for this endpoint is the same for every peer that tries to communicate with it. Consistency is usually easy to hole punch through, however some routers interact with other routers in ways that make it fail anyway.
  3. Predictable
    The publically visible port number for this endpoint is the same as the port that was opened on the local side. Predictability is not necessarily perfect, so connections to this host may fail depending on the types of networks involved.
  4. Impenetrable
    The publically visible port for this endpoint is entirely randomized. This means it can only connect to open ports.
  5. Unknown
    This address's state has not (yet) been determined.
